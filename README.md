EsCATe Game est un jeu de stratégie en ligne de commande (tour par tour) développé en Python sur une grille de 11x11. Deux joueurs s'affrontent : l'équipe des Chats (qui cherchent à s'échapper) contre l'équipe des Gardiens (qui tentent de les capturer), au milieu de robots autonomes et d'obstacles.

#Fonctionnalités clés
Mécaniques Chats : Jusqu'à 7 déplacements par tour et présence d'un "Super Chat" pouvant avancer de 2 cases.
Mécaniques Gardiens : Algorithme de saut par-dessus les obstacles (via calcul vectoriel) et pouvoir de téléportation limité.
Environnement dynamique : Entités neutres (Robots) avec IA de déplacement aléatoire.
Conception : Code architecturé en modules indépendants validés par des tests unitaires.

#Lancement
Le jeu s'exécute directement dans le terminal :
./application.sh
